### [완료] `next.config.ts` — picsum.photos 도메인 허용 추가
- **파일:** `next.config.ts`
- **원인:** 로컬 DB 테스트 데이터에 `picsum.photos` URL 이미지가 존재 → `PopularProducts.tsx`에서 `<Image src={img}>` 렌더링 시 Next.js가 미허용 도메인으로 Runtime Error 발생
- **배포판은 정상인 이유:** 배포 DB에는 실제 Supabase Storage URL만 존재 (이미 허용된 도메인)
- **수정 내용:**
  ```ts
  // next.config.ts remotePatterns에 추가
  {
    protocol: 'https',
    hostname: 'picsum.photos',
  }
  ```
- **영향 범위:** 로컬 개발 전용. 배포/Supabase/DB에 무관한 순수 Next.js 이미지 설정
